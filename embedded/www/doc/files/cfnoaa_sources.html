Description

Welcome to cf::noaa, a binding to the protobuf (un)marshalling C code for cf metron/doppler/noaa messages.

The audience of this document is anyone wishing to either have just a look at cf::noaa's source code, or build the packages, or to extend and modify them.

For builders and developers we additionally provide

  1. cf::noaa - License.

  2. cf::noaa - The Installer's Guide.

  3. cf::noaa - The Developer's Guide.

respectively.

Source Location

The official repository for cf::noaa can be found at http://github.com/ActiveState/cf-noaa-tcl

Retrieval

Assuming that you simply wish to look at the sources, or build a specific revision, the easiest way of retrieving it is to:

  1. Go to this site

  2. and use the standard github functionality

  3. to choose the revision you wish to have and

  4. get a copy of this revision in the format of your choice.

Source Code Management

For the curious (or a developer-to-be), the sources are managed by the Git SCM. Binaries for popular platforms can be found directly at its download page.

With that tool available the full history can be retrieved via:

    git clone git@github.com:ActiveState/cf-noaa-tcl.git

followed by

    cd cf-noaa
    git pull master

to get a checkout of the head of the trunk.

Bugs, Ideas, Feedback

This document, and the package it describes, will undoubtedly contain bugs and other problems. Please report such at the cf::noaa issue tracker. Please also report any ideas for enhancements you may have for either package and/or documentation.
