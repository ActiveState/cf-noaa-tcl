Description

Welcome to cf::noaa, a binding to the protobuf (un)marshalling C code for cf metron/doppler/noaa messages.

This document is the reference manpage for the publicly visible API, i.e. the API a user will see, and use. This API falls into two big sections, for the generation of encoded log messages from Tcl data structures on one side, and the parsing of encoded log messages into Tcl structures on the other.

Parsing encoded log messages

cf noaa unpack bytearray

This command parses the encoded log message in the Tcl bytearray and returns a Tcl dictionary where each key represents an element of the log message.

Generating encoded log messages

cf noaa pack message message_type timestamp ?app_id? ?source_type? ?source_instance?

This command converts the Tcl arguments representating the parts of a log message into a protobuf-encoded binary string and returns this bytearray as its result.

bytearray message

The message itself.

enum message_type

The message type, either out or err.

int64 timestamp

A 64bit timestamp for the message.

string app_id

An optional string.

string source_type

An optional string.

string source_instance

An optional string.
